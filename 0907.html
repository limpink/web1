전산 2

2021년 9월 7일

배운 점

  1. 표본 공간 (Sample Space)
    • Sample Space (S) : 가능한 모든 사건의 집합
  2. 사건 (Events)
    • Event (A) : 표본공간의 부분집합
    • Complement (A') : 여사건
    • Intersection (A ∩ B) : 교집합
    • Disjoint (A ∩ B = ∅) : 배반사건 (교집합 X)
    • Union (A ∪ B) : 합집합
  3. 경우의 수 (Couting Sample points)
    • Factorial : 팩토리얼

      n! = n(n-1) ⋯ 2 ⋅ 1 (n은 0 이상의 정수, 0! = 1)

    • Permutation : 순열
      • n명을 줄 세우는 방법은 n!
      • n명 중 r명을 줄 세우는 방법은 n! / (n - r)!
      • n명을 원탁에 앉히는 방법은 (n - 1)!
      • AABB를 재배열해 나오는 문자열은 총 4! / ( 2! * 2! )
      • 색종이 n장 중 같은 색깔이 각각 n1, n2, ⋯ , nm장 있을 때 이들을 올려놓는 방법은 n! / n1! n2! ⋯ nm!
    • Combination : 조합

      n명 중 r명을 뽑는 방법은 n! / r! (n - r)!
